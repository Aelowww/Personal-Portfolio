# Carl Gemuel Taberna Portfolio

Personal portfolio website built to present my projects, technical skills, certifications, and contact details in one place.

Live site: https://carldev.vercel.app

## Overview

This project is my personal developer portfolio. It highlights the work I have built, the tools I am currently using, and the direction I am growing toward as an aspiring full-stack developer.

## Tech Stack

- Next.js
- React
- JavaScript
- CSS
- Vercel Analytics

## Features

- Responsive landing page with project highlights
- Skills section with grouped technologies and proficiency levels
- Certificate pages and resume view
- Contact links and social profiles
- Built-in chatbot for quick portfolio questions
- Sitemap and SEO metadata for better discoverability

## Local Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Project Structure

- `app/page.jsx` contains the main landing page content
- `app/components/` contains reusable UI sections
- `app/api/chat/route.js` powers the portfolio chatbot
- `public/` stores portfolio images, gallery photos, and assets

## Purpose

I use this site to showcase my work, present my growth as a developer, and make it easy for recruiters, collaborators, and classmates to learn more about me.
